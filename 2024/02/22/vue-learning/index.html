Vue简介

  1. 渐进式框架
    灵活性，可以逐步集成，根据使用场景在不同页面使用
  2. vue3新版本
    涵盖vue2，新增体系
  3. 官方文档
    vue官方文档

Vue API风格

  1. 选项式API
    data methods mounted
  2. 组合式API（更推荐）
    代码量更少简洁，实现功能一样

开发前准备

  1. 下载nodejs注意版本
  2. 执行npm init vue@latest
  3. 进入项目执行npm init
  4. npm run dev 查看页面

Vue项目目录

  1. .vscode —vscode工具配置文件
  2. node_modules —Vue项目运行依赖文件 npm install下载的
  3. public —资源文件夹（游览器图标）
  4. src —源码 写代码
  5. .gitignore —git忽略文件
  6. index.html 入口html文件
  7. package.json —信息描述 版本依赖
  8. README.md —注释文件
  9. vite.config.js —Vue配置文件

Vue模板语法

  1. 文本插值
    双大括号引用对应的值
    仅支持唯一（单行if else不行）表达式，拥有return返回值
  2. 原始html
    想要插入html文件不使用双大括号，需要单独使用v-html插入
    双大括号会解释为纯文本
    <p v-html=""></p>
    v-html要写在第一个p中
